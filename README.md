# Social Media Impact Analysis – Instagram Usage & Wellbeing

## Project Overview
This project analyzes Instagram usage behavior and its relationship with user demographics, lifestyle patterns, and engagement metrics. The goal is to understand how time spent, content consumption, and engagement vary across different user segments and to translate raw social media data into actionable insights using SQL and Tableau.

## Business Problem
Social media platforms and digital marketers need to understand:

- Which user segments are most engaged
- How usage patterns differ by age group and lifestyle factors
- Whether excessive usage correlates with health and wellbeing indicators

This project simulates a real-world analytics scenario where behavioral data is analyzed to inform product strategy, digital wellbeing initiatives, and content optimization.

## Dataset
- Source: Public Instagram Usage Dataset
- Format: CSV files
- Level: User-level behavioral and demographic data

Key data elements analyzed:
- Time spent on Instagram
- Engagement metrics (likes, comments, messages)
- Employment status
- Content preferences
- Age groups
- Health and wellbeing indicators

## Tools & Technologies
- **SQL (PostgreSQL)** – Data cleaning, aggregation, KPI calculations
- **Tableau Public** – Interactive dashboards and visual storytelling
- **Excel** – Initial data validation and formatting
- **Tableau Calculated Fields** – Age grouping and engagement segmentation

## Data Preparation
The raw dataset was cleaned and standardized before analysis:

- Validated numeric fields such as age, time spent, and engagement metrics
- Removed inconsistent records
- Created derived analytical fields including:
  - Age groups
  - Engagement intensity
  - Average daily activity metrics

SQL queries were used to:

- Aggregate engagement metrics
- Calculate averages by demographic segment
- Identify high-engagement user cohorts

## SQL Analysis Examples
Examples of analytical queries performed:

- Count of users by age group
- Average daily Instagram usage by employment status
- Engagement metrics (likes, comments, DMs) per user
- Identification of high-usage vs low-usage user segments

SQL served as the analytical layer before visualization.

## Key Insights
- Younger age groups showed higher engagement frequency but shorter session durations.
- Employed users demonstrated more consistent but time-bound usage patterns.
- High engagement correlated with increased daily active minutes.
- Content preferences varied significantly across age segments.
- Excessive usage showed potential correlation with stress indicators (observational, not causal).
